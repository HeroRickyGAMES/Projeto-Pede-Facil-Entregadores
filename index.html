Política de Privacidade - Pede Facil - Entregadores

O app ele usa um banco de dados do Google chamado Firebase Database,

ele coleta as informações dos usuarios para te-las em ambiente seguro para o funcionamento do app!

O app, usa o GPS para determinar a localização do Entregador!

O app usa também a API de localização determinada para lojas também, porém com o uso apenas para determinações de endereços!

O app também usa a API de pagamentos da Stripe.com, uma plataforma 100% segura e pratica para o uso da loja e do entregador!
